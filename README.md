# Admin Dashboard (test)

## Overview
The Admin Dashboard provides a web-based interface for managing and querying the Smart Surveillance System.

It enables administrators to search events, track identities, monitor activity logs, and visualize real-time data.

## Core Features
- Secure authentication
- Event search and filtering
- Face lookup
- Timeline-based activity exploration
- Real-time monitoring interface
- System health visualization

## Why This Application Exists
The dashboard separates presentation logic from backend processing to:

- Enable independent frontend development
- Improve user experience iteration
- Maintain clean API-based architecture
- Support scalable deployments

## Technology Stack
- React or Next.js
- TailwindCSS
- JWT authentication
- REST API integration
- Optional WebSocket support

## Deployment
- Vercel / AWS S3 + CloudFront
- Connected to Cloud Backend API

## Future Improvements
- Role-based UI
- Advanced analytics visualizations
- Exportable reports
- Alert management interface
